Overview
--------

The Farm Animal Simulation is a GUI-based game where players can manage a virtual farm with various animals. The game features:

* Multiple animal types (Cow, Chicken, Sheep, Pig, Horse)
* Individual and group animal care options
* Animal breeding system
* Real-time status monitoring
* Random initial states for animals

Game Features
------------

* **Animal Management**
    * Feed individual animals
    * Play with individual animals
    * Feed all animals at once
    * Play with all animals at once

* **Breeding System**
    * Breed specific types of animals
    * Requires at least 2 animals of the same type
    * Creates new animals with unique names

* **Status Monitoring**
    * Health tracking
    * Hunger levels
    * Boredom levels
    * Age tracking

* **Game Controls**
    * Start/Pause functionality
    * Real-time updates
    * Status messages
    * Animal selection
